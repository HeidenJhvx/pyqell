# EGP v4.0 | Phase 5: Live Trading

---

## Phase 5 목표

Binance Futures 실거래 전환 및 장기 안정 운영.
EGP 임계점 자동 모니터링으로 리스크 관리.

---

## 임계 지표

```
완료 조건 1: 실거래 30일 이상 MDD < 10% 유지
완료 조건 2: 월 누적 수익률 양전환
완료 조건 3: EGP 임계점 위반 없음 (drawdown, signal_failure)
```
